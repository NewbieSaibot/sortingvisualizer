# Sorting visualizer

Sorting visualizer written in C using raylib and raygui

![Video](https://raw.githubusercontent.com/NewbieSaibot/sortingvisualizer/main/assets/thumb.jpg)(https://raw.githubusercontent.com/NewbieSaibot/sortingvisualizer/main/assets/sortingvisualizervid.mp4)

## Dependencies

`gcc`

`make`

`raylib`

## Building

`make`

## How to use

### Available buttons and input fields
Color
Pause / Resume
Reset / Apply

Speed (any positive integer number)
Array size (any integer number from 1 to 200000)

Sorting algorithm

### Binds
`C` --- Toggle color
`Space` --- Pause / Resume
`Key Right` --- Next iteration of sorting
`Esc` --- Quit

## Available sorting algorithms

`Bubble`

`Shaker`

`Gnome`

`Shell`

`Selection`

`Quick`

`Merge`

`Natural Merge`

`Heap`

`Bogo`
